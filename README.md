# Finance Dashboard (Final Submission)

## Features
- Dashboard overview (Balance, Income, Expense)
- Time-based (Line) and Category (Pie) charts
- Transactions table with search, filter, and sorting
- Role-based UI (Viewer/Admin with add form)
- Insights section (highest category, totals)
- CSV export
- Dark mode
- LocalStorage persistence
- Responsive layout with sidebar

## Tech Stack
- React + Vite
- Tailwind CSS
- Recharts
- Framer Motion

## Run
npm install
npm run dev

## Build
npm run build
